Guía de Instalación Paso a Paso

⚠️ PROBLEMA DETECTADO

Has intentado abrir index.html directamente y obtuviste un error de CORS:

"Access to fetch... has been blocked by CORS policy"

Esto es normal y tiene solución fácil.

✅ SOLUCIÓN RÁPIDA (30 segundos)

Necesitas ejecutar un servidor local. Es muy fácil:

1 Elige Tu Sistema Operativo

🪟 WINDOWS

Haz doble clic en el archivo:

servidor.bat

Se abrirá una ventana y el navegador automáticamente.

🍎 macOS / LINUX

Abre la Terminal en esta carpeta y ejecuta:

chmod +x servidor.sh
./servidor.sh

O simplemente haz doble clic en servidor.sh

2 El Servidor se Iniciará Automáticamente

Verás un mensaje como este:

✓ Servidor iniciado en: http://localhost:8000

┌────────────────────────────────────────┐
│ ABRE TU NAVEGADOR Y VE A: │
│ http://localhost:8000 │
└────────────────────────────────────────┘

3 Abre el Navegador

Si no se abre automáticamente, ve manualmente a:

http://localhost:8000

¡Ya no habrá errores de CORS!

❓ ¿Por Qué Necesito un Servidor?

Los navegadores modernos bloquean las peticiones a APIs externas cuando abres archivos HTML directamente (file:///) por razones de seguridad.

Un servidor local (aunque sea en tu propia computadora) resuelve este problema.

🔧 Método Manual (Si los Scripts no Funcionan)

Opción A: Python (Recomendado)

1. Verifica si tienes Python instalado:

python --version

2. Si no lo tienes, descárgalo desde: python.org

3. Abre una terminal en la carpeta del juego y ejecuta:

python -m http.server 8000

4. Abre tu navegador en: http://localhost:8000

Opción B: Node.js

1. Verifica si tienes Node.js instalado:

node --version

2. Si no lo tienes, descárgalo desde: nodejs.org

3. Ejecuta en la carpeta del juego:

npx http-server -p 8000

4. Abre tu navegador en: http://localhost:8000

🎯 Una Vez que el Servidor Esté Funcionando

  1. Ve a console.anthropic.com
  2. Obtén tu API Key
  3. En el juego, ingresa tu API Key
  4. Haz clic en "Probar Conexión"
  5. Si funciona, haz clic en "Guardar API Key"
  6. ¡Comienza a jugar! ♟️

💡 CONSEJO IMPORTANTE

NO cierres la ventana del servidor mientras estés jugando. Si la cierras, el juego dejará de funcionar.

Para detener el servidor cuando termines: Ctrl+C en la ventana del terminal.
