# Chess Piece Sets

Este directorio contiene varios sets de piezas de ajedrez en formato SVG, descargados de Lichess.org.

## Sets Disponibles

### 1. **Cburnett** (cburnett/)
- Estilo clásico y elegante
- El set más popular en Lichess
- Creado por Colin M.L. Burnett
- Licencia: GPL

### 2. **Mérida** (merida/)
- Estilo tradicional y profesional
- Basado en el diseño Mérida
- Muy usado en torneos

### 3. **Pixel** (pixel/)
- Estilo retro de pixel art
- Perfecto para un look retro/8-bit
- Diseño minimalista

### 4. **Fantasy** (fantasy/)
- Estilo artístico con detalles decorativos
- Piezas con más ornamentación
- Aspecto medieval

### 5. **Letter** (letter/)
- Estilo de letras tipográficas
- Simple y funcional
- Similar al estilo alpha pero con SVG

## Nomenclatura de Archivos

Cada pieza sigue este formato: `[color][tipo].svg`

- **Color**: `w` (white/blanco) o `b` (black/negro)
- **Tipo**: `K` (King), `Q` (Queen), `R` (Rook), `B` (Bishop), `N` (Knight), `P` (Pawn)

Ejemplos:
- `wK.svg` = Rey blanco
- `bQ.svg` = Reina negra
- `wP.svg` = Peón blanco
